ADW Data Files

This data file format comes from previous DOCoMETRe version 2.
It is a binary file with headers containing all experiment informations and samples. There must be this kind of data file for each subject.
You can read below a pseudo code which can be used to read such a file :

Read number of channels (C) (int32 size)
Read all channels names separated by pipe ('|') character (sequence of characters : C names to read)
Read number of sessions (S) (int32 size)
Read Sessions names separated by pipe ('|') character (sequence of characters : S names to read)
Read number of conditions (Co) (int32 size)
Read Conditions names separated by pipe ('|') character (sequence of characters : Co names to read)
Read number of 'sequence type' (St) (int32 size)
Read 'Sequence type' names separated by pipe ('|') character (sequence of characters : St names to read)
Read total number of trials (int32 size)
For each of these trials :
Read session number (int32 size)
Read condition number (int32 size)
Read sequence type number (int32 size)
Read channels number (int32 size)
For each channel in this trial :
Read channel number (int32 size)
Read sample frequency float32
Read samples number (N) (int32 size)
Read samples values (N*float32 size)
