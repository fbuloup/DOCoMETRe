Samples Data Files

This is the default data file format. It is a binary file containing samples values.
The file name must at least hold the channel name and the trial number. It can also contain a prefix and a session name.

Here are two examples of such a file name : 

 prefix.ChannelName.SessionName.T32.samples
 ChannelName.T32.samples

You can find below a Matlab code to read this kinf of file :

fileHandle = fopen(dataFileName, 'r');
data = fread(fileHandle, 'float32')';       
fclose(fileHandle);

And below a Python code to read this kind of file using Numpy library :

 data = numpy.fromfile(dataFileName, dtype="float32");
