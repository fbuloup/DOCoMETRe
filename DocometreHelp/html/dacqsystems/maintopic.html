Introduction

For now you can use ADWin and Arduino Uno systems.

ADWin systems are declined in several hardwares :
  • ADWin Gold
  • ADWin Gold II
  • ADWin Pro
  • ADWin Pro II
All these hardwares are supported by DOCoMETRe.

Note that ADWin systems are mainly available on Windows. They are supposed to work under Linux but if you want to use these dacq. systems under Mac OS X, you must install WineHQ.

Warning : on Mac OS X, WineHQ is no longer available since Mac OS version 10.15.

Arduino Uno R3 and R4 are fully supported on Windows, Mac and Linux platforms. ADS1115 extension module is also supported.

Support for MCP4725 (DAC 12 bits - I2C) Arduino Uno extension module is planned. Support for Portenta Machine Control or Teensy 4.0 support are also planned.
