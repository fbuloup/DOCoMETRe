Troubleshoots with Arduino Uno System

I am on Linux and can't see Arduino Uno port. What can I do ?

On linux OS, an Arduino Uno is usually connected to /dev/ttyACMn. But if you do not have right access to /dev/ttyACMn, you won't be able to use it in DOCoMETRe. In order to access this device, please run these commands in terminal : 

   sudo usermod -a -G tty UserName
   sudo usermod -a -G dialout UserName
   sudo chmod a+rw /dev/ttyACMn

These commands simply add UserName to tty and dialout groups and make sure UserName has read and write acces to /dev/ttyACMn. Please follow this link if you need more informations. You must obviously use proper values for UserName and n.

My Arduino Uno processes using ADS1115 module doesn't compile.

If you get this kind of error message :

fatal error: ADS1X15.h: No such file or directory
 #include "ADS1X15.h"
                     ^
compilation terminated.
exit status 1
Erreur de compilation pour la carte Arduino/Genuino Uno

It means that you need to install ADS1115 driver from Rob Tillaart as build-in Arduino Uno Library. To do so, please follow this github link : ADS115 Library. Then click on Download Button and choose Download ZIP Menu Item. Once Zip file has been downloaded and uncompressed, rename corresponding folder to ADS1115 and cut/past it in Java/libraries folder of your Arduino Uno IDE installation directory. You can also install this library from Arduino IDE direclty. If you choose this option, please follow this link to find a tutorial on installing additional libraries : Installing Additional Arduino Libraries. Compilation should now occur without any error.
