How to open samples data files with Matlab or Python ?

It's possible to open samples data files created by DOCoMETRe. These files are simply binaries files using IEEE754 floating point standard.

You can find below a Matlab code to read this kinf of file :

fileHandle = fopen(dataFileName, 'r');
data = fread(fileHandle, 'float32')';       
fclose(fileHandle);

And below a Python code to read this kind of file using Numpy library :

 data = numpy.fromfile(dataFileName, dtype="float32");
