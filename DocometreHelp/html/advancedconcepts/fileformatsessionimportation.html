What is file format for sessions importation ?

It's possible to create a text file containing a predefined protocol. This text file must have ".ini", ".txt" or ".properties" extension. It can contain comments using "#" character.

Here is an example of such a file : 

# WARNING : Java properties keys are case sensitive. 
# Therefore each property key must be LOWER case
# Sessions names must respect this regular expression : ^[a-zA-Z][a-zA-Z0-9_]*$
# Trial properties are "startMode:pathToAssociatedProcessFile"
# Trial start mode possible values are : auto/manual
# Associated process file path must be relative to workspace root.
# If this relative path does not exist in workspace, it won't be taken in account. 

# Session 1
session1.name=session3
session1.trial1 = manual:/ExempleArduino/pArduino.process
session1.trial2 = manual:/ExempleArduino/pArduino2.process
session1.trial3 = manual:/ExempleArduino/pArduino.process
