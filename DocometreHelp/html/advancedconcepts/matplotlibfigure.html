Is it possible to use python Matplotlib or Matlab figures ?

It's possible to add expressions functions that contain Matlab or Python code to display data. You will find below two examples in both languages.

Display data with Matlab :

 
close all;
values = DA.SUJET1.CODAmotion_C1_X.Values; % Get all trials samples
nbSamples = length(values(1,:)); % First (or any) trial length is number of samples 
sampleFrequency = DA.SUJET1.CODAmotion_C1_X.SampleFrequency; % Get signal sample frequency
time = (0:nbSamples-1)/sampleFrequency; % Create time vector
% figure_handle = figure; % if you want to make figure window to be modal
% set(figure_handle,'WindowStyle','modal')
plot(time, values(1,:)); % Plot first trial
shg % Make figure to be displayed on top

Display data with Python using Matplotlib :

import matplotlib.pyplot as plt; # Import pyplot
import numpy as np; # Import numpy
values = D["DA.SUJET1.CODAmotion_C1_X.Values"]; # Get all trials samples
nbSamples = len(values[1,]); # First (or any) trial length is number of samples 
sampleFrequency = D["DA.SUJET1.CODAmotion_C1_X.SampleFrequency"]; # Get signal sample frequency
time = np.r_[0.:nbSamples]/sampleFrequency; # Create time vector
plt.plot(time, values[0,:]); # Plot first trial
plt.plot(time, values[1,:]); # Plot second trial
plt.ylabel('First and second trial'); # Some y label
plt.show();
Note that D is a shortcut for docometre.experiments
